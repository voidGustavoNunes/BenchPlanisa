Sobre o Projeto

Esta aplicação foi desenvolvida como uma ferramenta de análise comparativa de dados da COVID-19 entre duas localidades. Criada utilizando Spring Boot para o back-end e Angular para frontend, a aplicação permite realizar cadastros, edições e remoções de benchmarks, fornecendo informações detalhadas sobre casos confirmados, óbitos e outros dados relevantes.

Os benchmarks são criados a partir da inserção de duas localidades e um período específico, utilizando os dados obtidos da API pública Brasil.IO. Esses dados são armazenados no banco de dados da aplicação, permitindo que sejam consultados e analisados posteriormente.

Além disso, a aplicação disponibiliza uma API que retorna os dados comparativos com base no ID do benchmark ou através da especificação duas localidades e do período desejado.
